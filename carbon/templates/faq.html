Frequently Asked Questions

Drunk on Carbon is a final project for a HarvardX computer science course. See the About page.

We use the Climatiq API, which allows us to access a database of up-to-date and scientifically vetted emission factors.

When you enter a number of miles, we send that number straight to Climatiq.
Climatiq sends back the kg of CO2e.

When you enter an origin and destination, we use two Google Maps APIs to calculate the number of miles.

First we send the origin and destination to the Google Maps Places API.
Google sends back a unique Google ID and formatted address for each one.

Then we send the place IDs to the Google Maps Distance Matrix API.
Google sends back the distance between those two places in meters.

We convert the meters to miles, then send that value to the Climatiq API.
